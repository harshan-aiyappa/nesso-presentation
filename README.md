# Nesso Botanica — Ayurvedic Intelligence Presentation

A high-fidelity, interactive 11-slide presentation for **Nesso Botanica**, showcasing the power of Ayurvedic Graph Intelligence. This project is built with 100% literal content fidelity and premium web aesthetics.

## ✨ Key Features

- **Character-Perfect Content**: All slides (1-12, Slide 11 removed) restored with literal precision.
- **Micro-Animations**: 
  - **Magnetic Hover**: Elements gently drift toward the mouse cursor for a tactile feel.
  - **3D Parallax**: Background mesh shifts with mouse movement for added depth.
  - **Adaptive Interaction**: Modern ring-cursor on desktop; completely native touch experience on mobile/tablets.
- **Cinematic Flows**: Slide 9 features a sequential, looping animation representing the "Ask → Notified" intelligence lifecycle.
- **Premium Aesthetics**: Glassmorphism, Deep Emerald & Gold branding, and optimized GSAP staggers.
- **Deployment Ready**: Standardized configurations for Vercel and Netlify included.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Modern layouts with Flexbox/Grid and Glassmorphism.
- **GSAP (GreenSock)**: Professional-grade animations and ScrollTrigger.
- **Git**: Version controlled and deployment optimized.

## 🚀 Getting Started

### Local Development
1. Clone the repository.
2. Open `index.html` in your browser (Recommended: use a local server like Live Server for the best experience with animations).

### Deployment
This project is optimized for **Vercel** and **Netlify**.

#### Vercel (Recommended)
1. Push to your GitHub repository.
2. Import the project in Vercel.
3. The included `vercel.json` handles all configurations automatically.

#### Netlify
1. Connect your GitHub to Netlify.
2. The `netlify.toml` file will handle the static deployment settings.
